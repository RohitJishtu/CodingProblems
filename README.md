# Coding Problems Collection

This repository contains a collection of coding problems and their solutions, organized into various categories and modules. The problems come from various sources including LeetCode, Coding Ninjas, Educative, and interview questions from companies like PNC, FAANG, and C3AI.

## Repository Structure

The repository is organized into several main directories:

- **Coding problems & Solution**: Contains solutions to various coding problems
  - **Educative**: Problems from Educative platform
  - **CodingNinjas**: Problems from CodingNinjas platform
  - **CodingPatternsV2**: Problems organized by coding patterns
  - **DocsCode**: NEET series problems
  - **DataAnalysis**: CSV reading and analysis problems

- **CodingQuestions**: Contains mostly LeetCode problems
- **Concept_Problems**: Contains implementations of various concepts and data structures
- **LinkedList**: LinkedList implementations
